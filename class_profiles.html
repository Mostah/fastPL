bool Profiles::isProfileOrdered ( )

isProfileOrdered check if the performance table is ordered to be use as a profile type: each row must be ranked such that for each row i we have on each criterion j: v_i-1_j > v_i_j > v_i+1_j. Profiles are therefore ordered in a descending manner
